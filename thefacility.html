  • Overview

    The Facility is a Laboratory themed escape room built in Unity3D. It was a solo project, made during a UX focused module at the University of Plymouth. This module focused on my ability to design an immersive and coherent user experience, with appropraite documentation and usability testing to back it.

    The Facility features 3 puzzles, each of which have randomly generated answers each time the user goes through a game sequence. Puzzle one takes a random word from a large list, and then generates the puzzle automatically from that string. Puzzle two randomises the door codes every time the user plays, and puzzle three does the same.

  • The Design Process

    "Welcome to The Facility."

    This project started with an initial prompt, 'Escape Room'. I started by researching the possibilities of an escape room, and mindmapping all of the various connotations that term may have. I had been replaying Portal 2 during ths start of this project, so I was really inspired to create a 'laboratory' themed escape room, much like those seen in Portal 2.

    I took this further into a user story, laying out the experience I want the player to have while they go through the game. I could then start making moodboards, which allowed me to develop the environment style, lighting and puzzles. This is when I took things over to paper, sketching out each puzzle in a notebook while whiteblocking the functionality and layout in Unity (see left). I found it really helpful to be able to plan a room layout, then immediately block it into a 3D environment.

    The Facility was my first introduction to a formalised design process, and I believe is a great example of how I can develop initial concepts into working prototypes.
